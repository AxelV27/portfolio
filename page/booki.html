Booki

Mon tout premier projet de ma formation, ici je suis un développeur web en alternance au sein de la start-up Booki. L'idée était d'intégrer une première version du site via une maquette figma qui était fournis pour permettre aux utilisateurs de trouver un hébergement et une activité. Il m'a permis de découvir les bases fondamentales du code.

Compétences

HTML

Ce projet utilise de l'HTML, il m'a appris les bases du code avec les balises sémentiques

CSS

Ce projet utilise du CSS, il m'a appris les bases avec la disposition des éléments en grid

Responsive design

Ce projet dispose une version tablette et téléphone m'apprennant le responsive design
